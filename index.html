Scope

This dashboard focuses on Active Phase 3 interventional trials for the global top‑20 pharmaceutical companies by market capitalization (as of Jan 2026).

• **Strict Filter:** Pure Phase 3 Only (Excludes Ph 1, 1/2, 2, 2/3)
• **N ≥ 100 patients:** To exclude very small studies

**Included statuses:** Recruiting, Active (not recruiting), Enrolling by invitation, and Not yet recruiting.
**Rationale:** The filters are tuned to highlight large, registration‑intent Phase 3 programs rather than exploratory or post‑marketing studies.

Scientific Methodology

Clinical trial data is aggregated via high-fidelity synchronization with the ClinicalTrials.gov API v2. The "Pure P3" filter is a specialized heuristic designed to separate registrational-intent studies from earlier exploratory research.

Primary Data Sources:
- *U.S. National Institutes of Health. ClinicalTrials.gov API v2 (Sync: Feb 2026).*
- *S&P Global / CompaniesMarketCap. Sector rankings (Jan 2026).*
- *WHO International Clinical Trials Registry Platform (ICTRP).*
